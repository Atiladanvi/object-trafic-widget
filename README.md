# vue-object-trafic

## Project setup
```
npm install vue-object-trafic
```

###  Import and use
In your main.js 
```javascript
import Vue from 'vue';
import ObjectTrafic from 'vue-object-trafic'

Vue.use(ObjectTrafic , { lang: 'pt' })
```
In your component
```vue
 <object-trafic-lite :data="[1,2,3,4]"></object-trafic-lite>
```
